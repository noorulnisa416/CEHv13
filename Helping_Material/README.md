## 3Ô∏è‚É£ OSI Model

![OSI Model](https://upload.wikimedia.org/wikipedia/commons/9/9a/OSI_Model_7_Layers.svg)  

The **OSI model** is a framework to understand network communications. It has **7 layers**:

| Layer | Function | Examples/Protocols |
|-------|---------|------------------|
| Physical | Transmits raw bits | Ethernet, USB |
| Data Link | Error detection & MAC addressing | Switches, ARP |
| Network | Routing & IP addressing | IP, ICMP |
| Transport | Reliable data transfer | TCP, UDP |
| Session | Manages sessions | RPC, NetBIOS |
| Presentation | Data translation & encryption | SSL, JPEG, ASCII |
| Application | End-user services | HTTP, FTP, DNS |

Understanding OSI layers helps in **troubleshooting networks** and analyzing **cybersecurity threats**.

## 4Ô∏è‚É£ TCP/IP Suite

![TCP/IP](https://upload.wikimedia.org/wikipedia/commons/5/5e/Tcp_ip_model_diagram.png)  

The **TCP/IP model** is used in real-world networking. Its layers are:

| Layer | Function | Protocols |
|-------|---------|-----------|
| Network Interface | Physical & Data Link | Ethernet, Wi-Fi |
| Internet | Routing & Addressing | IP, ICMP |
| Transport | Reliable delivery | TCP, UDP |
| Application | User services | HTTP, HTTPS, FTP, SMTP, DNS |

**Common ports:**
- HTTP ‚Üí 80 üåê
- HTTPS ‚Üí 443 üîí
- FTP ‚Üí 21 üìÇ
